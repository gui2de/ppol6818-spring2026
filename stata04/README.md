# README: De-Biasing Parameter Estimates Using Controls

## 1. Project Overview
This project explores how different types of covariates (confounders, mediators, and colliders) affect the bias and convergence of an OLS regression parameter estimate ($\beta$) as the sample size ($N$) grows. 

Using a custom Data Generating Process (DGP) in Stata, we simulated data where the true total causal effect of a treatment variable on an outcome variable $Y$ is fixed at **$0.3$ standard deviations**. We then observed how five different regression specifications behaved across multiple sample sizes ($N = 50$, $N = 500$, and $N = 5000$).

---

## 2. Data Generating Process (DGP)
The simulated dataset was constructed using basic operations and standard normal random variables to establish clear causal pathways. The variables were generated as follows:

* **Confounder ($C$):** Drawn from a standard normal distribution. It directly influences both the likelihood of receiving the treatment and the final outcome $Y$. 
* **Treatment ($T$):** Driven by the confounder plus a random shock ($u_t$).
* **Mediator ($M$):** Directly caused by the treatment variable plus a random shock. It represents a causal mechanism through which treatment influences $Y$.
* **Outcome ($Y$):** The ultimate variable of interest. It is directly generated by the treatment, the confounder, the mediator, and a standard normal error term ($u_y$). The pure direct effect is set at $0.1$, and the indirect effect passing through the mediator contributes another $0.2$, summing up to a **true total effect of $0.3$**.
* **Collider ($K$):** Generated as a direct function of *both* the treatment and the outcome $Y$, plus a random error term. 

---

## 3. Regression Models
To observe the behaviors of these variables, we constructed five distinct OLS regression models using different combinations of covariates:

| Model | Specification | Description / Expected Behavior |
| :--- | :--- | :--- |
| **Model 1** | `regress y treat` | **Raw (Omitted Variable Bias).** Captures a false inflated correlation because it fails to control for the background confounder. |
| **Model 2** | `regress y treat confounder` | **Ideal Model.** Successfully blocks the backdoor path from the confounder, yielding the true total treatment effect of $0.3$. |
| **Model 3** | `regress y treat confounder mediator` | **Over-controlled.** By controlling for the mediator, it blocks a valid causal mechanism, only showing the isolated direct effect ($0.1$). |
| **Model 4** | `regress y treat confounder collider` | **Collider Bias.** Controlling for the collider induces a fake, highly biased correlation between treatment and outcome. |
| **Model 5** | `regress y treat confounder mediator collider`| **Kitchen Sink.** Controls for everything. Heavily biased due to the persistent inclusion of the collider. |

---

## 4. Results & Convergence 

### 4.1 Summary Table of Beta Estimates
Below are the averaged results across $200$ Monte Carlo simulations for each specified sample size. 

| Model | $N = 50$ (Mean $\beta$) | $N = 500$ (Mean $\beta$) | $N = 5000$ (Mean $\beta$) |
| :--- | :---: | :---: | :---: |
| **Model 1: Raw** | $0.758$ | $0.755$ | $0.758$ |
| **Model 2: Ideal** | $0.598$ | $0.601$ | $0.602$ |
| **Model 3: Mediator Blocked** | $0.284$ | $0.291$ | $0.289$ |
| **Model 4: Collider Bias** | $-0.015$ | $-0.005$ | $-0.003$ |
| **Model 5: All Variables** | $-0.141$ | $-0.133$ | $-0.130$ |

*(Note: In this specific simulation, Model 3 reflects the pure direct effect of the treatment isolated from the mediator. Model 2 reflects the total mathematical effect inclusive of indirect pathways).*

### 4.3 Key Findings from the Visuals
1. **Variance Shrinkage:** As $N$ grows from $50$ to $5000$, the variance of the estimates shrinks aggressively. The lines become entirely flat at $N = 5000$, representing highly precise convergence on the expected expected value for that specific model.
2. **The Peril of the "Kitchen Sink" Approach:** Models 4 and 5 prove that "adding more controls" does not make a regression more accurate. Controlling for a collider fundamentally broke the model, pushing the estimated beta below zero when it is actually positive.
3. **The Importance of Model Selection:** Only specifications derived from accurate causal assumptions can effectively isolate the parameter of interest.
